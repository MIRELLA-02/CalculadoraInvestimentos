# CalculadoraInvestimentos

# 📄 Descrição
Uma aplicação desenvolvida em Flutter para simular investimentos, permitindo calcular o valor final com e sem juros. O usuário informa o valor do investimento, o tempo e a taxa de juros, e o app retorna o montante total sem rendimento e com rendimento, facilitando a comparação e o entendimento dos ganhos.
